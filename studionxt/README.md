# StudioNXT — Intelligent Archive for Artists

**Gemini Live Agent Challenge Submission — Creative Storyteller Category**

StudioNXT is an AI-powered digital archive that helps artists document, understand, and preserve their life's work. Built for Carol — a 94-year-old artist with 60+ years of creative output.

---

## The Problem

Every serious artist faces the same crisis: decades of work scattered across studios, storage units, galleries, and collections — with no coherent record. When an artist dies, their estate loses works, undervalues others, and cannot tell the story of a life's practice.

## The Solution

StudioNXT combines Google Gemini AI with a structured archive to create a living record of an artist's work — one that survives them.

---

## How Google AI Powers StudioNXT

### Gemini 1.5 Flash — Creative Storytelling
- **Artist biography generation** — Gemini writes a gallery-quality biography from the artist's archive data
- **Artist statement generation** — Gemini synthesises the artist's practice into a professional statement
- Route: `src/app/api/gemini/route.ts`

### Google Cloud — Firebase Platform
- **Firestore** — stores all artwork records, edition ledgers, voice sessions, legacy contacts
- **Firebase Storage** — stores original high-resolution artwork images
- **Firebase Authentication** — magic link login for artists
- **Firebase Hosting** — web deployment
- Project: `studionxt-2657b`

---

## Features

- **Archive** — Upload artworks with medium, dimensions, materials, edition data
- **Edition Ledger** — Track every physical copy: Unique / Limited / Open edition, AP tracking
- **Location Intelligence** — Know where every work is: Studio, Gallery, Collector, Storage, Museum
- **Valuation Engine** — AI-powered pricing based on career stage, market, scarcity
- **Voice Sessions** — Record Carol talking about her work with Mira
- **Legacy Contact** — Designate who receives the archive
- **Artist Biography** — Generated by Gemini from archive data
- **The Mira Letter** — Living document of the artist's practice (coming soon)

## Reproducible Testing Instructions

### Prerequisites
- Node.js 18+
- A Firebase project
- A Gemini API key from https://aistudio.google.com/apikey
- An Anthropic API key from https://console.anthropic.com

### Setup

1. Clone the repository
```bash
git clone https://github.com/artnxtstudio/StudioNXT.git
cd StudioNXT/studionxt
```

2. Install dependencies
```bash
npm install
```

3. Create `.env.local` with your keys
```
NEXT_PUBLIC_FIREBASE_API_KEY=your_firebase_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.firebasestorage.app
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
GEMINI_API_KEY=your_gemini_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key
```

4. Run the development server
```bash
npm run dev
```

5. Open http://localhost:3000

### Testing Gemini Features

1. Go to **Profile** page
2. Click **Generate Bio with Gemini**
3. Watch Gemini write a gallery-quality artist biography
4. The API call goes to `src/app/api/gemini/route.ts`
5. Uses Gemini 1.5 Flash model

### Testing the Archive

1. Click the **+** button
2. Upload an artwork photo
3. Fill in title, year, medium
4. Select classification (Unique / Limited Edition / Open Edition)
5. Set location (Studio, Gallery, Storage, etc.)
6. View the artwork detail with accordion sections

### Testing Legacy Contact

1. Go to **Profile**
2. Scroll to **Legacy Contact** (gold section)
3. Click **Designate legacy contact**
4. Fill in name and relationship
5. Save — archive is now protected
